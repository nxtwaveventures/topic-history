# OpenAI

## Origins (2015 - 2018)
- 2015: Founded as a nonprofit by Altman, Musk, Brockman, Sutskever and others
- Mission: ensure AGI benefits all of humanity
- 2018: Elon Musk leaves the board
- 2018: GPT-1, the first Generative Pre-trained Transformer

## Scaling era (2019 - 2021)
- 2019: "Capped-profit" structure; first $1B Microsoft investment
- 2019: GPT-2, released in stages over misuse fears
- 2020: GPT-3 and the API; few-shot learning goes mainstream
- 2021: Codex powers GitHub Copilot; DALL-E 1

## ChatGPT explosion (2022 - 2023)
- Nov 2022: ChatGPT launches, ~100M users in two months
- 2023: GPT-4, multimodal; ChatGPT Plus and plugins
- Nov 2023: Altman fired then reinstated within 5 days; board overhauled

## Productization and rivalry (2024)
- GPT-4o (omni: text, voice, vision)
- Sora video; o1 reasoning models
- Rivalry hardens: Anthropic Claude, Google Gemini

## Agents and enterprise (2025 - 2026)
- Codex CLI becomes the flagship developer agent
- GPT-5 model family
- Apr 2026: Microsoft exclusivity loosened (non-exclusive license through 2032)

## Now (last 30 days)
- Codex everywhere: ChatGPT mobile app, AWS Bedrock GA (Jun 1), Codex "Sites"
- GPT-5.5 framed as an agent model (1M-token reasoning 36.6% to 74.0%)
- Biggest ChatGPT "superapp" overhaul since launch, staged pre-IPO
- CFO Sarah Friar: record $122B raise; IPO race vs Anthropic's S1 filing

## Where it's heading
- $1T+ IPO talk for late 2026
- Compute is the gating constraint ("$100B = how many gigawatts?")
- Shift from chatbot to agent + enterprise platform
- Polymarket: frontier status near-certain (97%), but benchmark milestones cooling
